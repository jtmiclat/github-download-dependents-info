# Github Download Dependents Info

Download a repo's "Used by" page as a CSV

## Usage

```
pip install bs4 requests
python dl-dependency.py <USER>/<repo>
```

## Example

```
python main.py fastai/nbdev
python main.py fastai/nbdev --csv file.csv
```
